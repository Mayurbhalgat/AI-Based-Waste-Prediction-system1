# 🍽️ Smart Kitchen — AI Food Ingredient Predictor

AI-based system that predicts ingredient quantities,
detects food waste and alerts NGOs automatically.

## How to Run
1. Download all files from this repo
2. pip install -r requirements.txt
3. python app.py
4. Open http://localhost:5000

## Note
Download best_model.keras separately from Google Drive
and place it in the same folder as app.py
Link: YOUR_GOOGLE_DRIVE_LINK_HERE
```

4. Replace `YOUR_GOOGLE_DRIVE_LINK_HERE` with your actual Google Drive link
5. Click **Commit changes**
```
